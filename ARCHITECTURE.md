# AI Spend Audit — Architecture

## Overview

AI Spend Audit is a web application designed to help users analyze and optimize their AI subscription spending. The platform provides cost visibility, pricing comparisons, and recommendations for reducing unnecessary expenses.

The application is built using Next.js and deployed on Vercel.

---

# Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js |
| Styling | Tailwind CSS |
| Deployment | Vercel |
| Version Control | GitHub |
| Package Manager | npm |
